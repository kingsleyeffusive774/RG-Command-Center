How the pipeline works

Locked flow
1. Raw source records land in source_a, source_b, or manual_uploads.
2. Reconcile merges duplicates, flags source inconsistencies, and builds canonical internal listings.
3. Compile generates the public release pack. The public site never reads raw intake directly.
4. Fully local operation: no server dependency is required for data browsing, internal review, or public release compilation.
